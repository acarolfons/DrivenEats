# 🍔 DrivenEats

Aplicação web de delivery de comida, desenvolvida como primeiro projeto prático do Módulo 4 do curso **Driven Education**, com foco em HTML, CSS e JavaScript🎉

## 🔗 Deploy

Acesse o projeto online: [driven-eats-six.vercel.app](https://driven-eats-six.vercel.app)

---

## 📋 Sobre o Projeto

O DrivenEats é uma página de delivery inspirada em aplicativos como iFood, onde o usuário pode navegar por um cardápio, selecionar itens e montar seu pedido. O projeto foi desenvolvido com foco em boas práticas de front-end, responsividade mobile e manipulação do DOM com JavaScript puro.

---

## 🚀 Funcionalidades

- Listagem de categorias e produtos do cardápio
- Seleção de itens para o pedido
- Cálculo dinâmico do valor total do carrinho
- Finalização de pedido via WhatsApp
- Layout responsivo para dispositivos móveis

---

## 🛠️ Tecnologias Utilizadas

| Tecnologia | Uso |

| HTML5 | Estrutura da página |
